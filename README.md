# Pokédex

Click here to check my [Pokédex](https://pokedex-orpin-two.vercel.app/)! Try it in different devices :)

## About
This is a project to use JS to consume an API, in this case, the [PokéAPI](https://pokeapi.co/)!
You can list all the known Pokémons and search for one of them using their name or number.

## Main JavaScript concepts used in the project:

- array manipulation
- document.getElementById()
- document.querySelector()
- document.createElement()
- element.textContent
- element.append()
- element.remove()
- element.setAttribute()
- element.setProperty()
- addEventListener()
- object manipulation
- DOMPurify
- switch case
- rest API consumption
- functions
- destructuring
- IntersectionObserver
- Pormise, async, await
- fetch()
- try, catch
